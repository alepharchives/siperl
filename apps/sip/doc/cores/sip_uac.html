create_request/3

create_request(Method::atom() | binary(), Sip_hdr_address::#sip_hdr_address{}, Sip_hdr_address::#sip_hdr_address{}) -> #sip_message{}

Create request outside of the dialog according to the 8.1.1 Generating the Request

Creates all required headers of the SIP message: Via:, Max-Forwards:, From:, To:, CSeq:, Call-Id, Contact:. Also, adds Route: headers if pre-existing route set is configured.

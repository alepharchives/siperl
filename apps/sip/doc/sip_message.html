header_values/2

header_values(Name::atom() | binary(), Message::#sip_message{} | [{Name::atom() | binary(), Value::binary() | term()}]) -> [any()]

Retrieve values for given header name

Retrieve value of given header. Accepts either full SIP message or list of headers. Values of several headers with the same name are merged into single list. Note: this function parses the header value if header is in binary form. Note: this function is safe to use only with headers that allow merging multiple values together. For example, Authentication-Info or Contact header with value * do NOT allow this). See RFC 3261, Section 7.3

update_top_header/3

update_top_header(HeaderName::atom() | binary(), Fun::fun((Value::any()) -> UpdatedValue::any()), Sip_message::#sip_message{}) -> #sip_message{}

Update value of top header with given name.

If header value is multi-value, only first element of the list (top header) is updated by the function.

Note: this function parses the header value if header is in binary form. If no header is found with given name, update function is called with undefined parameter. If function returns any value other than undefined, a new header with that value is added.

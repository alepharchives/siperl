Description

Pipeline monad

Basically, this monad is equivalent to error_m monad provided by Erlando. The only difference is more meaningful names. {next, T}, which is equivalent of {ok, T} is to invoke next operation. {stop, Result}, which is equivalent of {error, Result} is to stop computation and return result.
